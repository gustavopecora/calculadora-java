# 🧮 Calculadora em Java

Projeto simples de calculadora desenvolvido em Java para praticar lógica de programação.

## 🚀 Funcionalidades
- Soma
- Subtração
- Multiplicação
- Divisão
- Validação de divisão por zero
- Loop para múltiplas operações

## 💻 Tecnologias
- Java
- Eclipse

## 📚 Aprendizados
- Estrutura de decisão (if/else)
- Estrutura de repetição (do-while)
- Entrada de dados com Scanner
- Lógica de programação

👨‍💻 Projeto feito para evolução como desenvolvedor.
